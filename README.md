# 🔥 Portfolio Website - Imu Abdurroofi

Portfolio website modern dengan tema **Dark + Red** yang menampilkan profil, skills, projects, dan kontak.

## ✨ Fitur Utama

### 🎨 Desain Modern
- **Tema**: Dark mode dengan aksen merah
- **Style**: Clean, futuristik, profesional
- **Effects**: Red glow, shadows, smooth animations

### 🧭 Sidebar Navigation
- Fixed sidebar di kiri
- Active state highlighting
- Smooth scroll ke section
- Icon-based menu (Font Awesome)
- Responsive hamburger menu (mobile)

### 🎬 Animasi
- Fade in saat scroll (Intersection Observer)
- Hover effects pada cards
- Smooth scrolling
- Loading animation
- Parallax effect

### 📱 Responsive Design
- Mobile-first approach
- Sidebar → Hamburger menu (≤1024px)
- Flexible grid layouts
- Touch-friendly interface
- Optimized untuk semua device

### 🎵 Background Music
- Music player dengan toggle button
- Auto-loop
- Volume control (30%)
- Visual feedback (spinning animation)

## 🛠️ Teknologi

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, Animations
- **JavaScript (ES6+)** - Vanilla JS, no framework
- **Font Awesome** - Icons
- **Intersection Observer API** - Scroll animations

## 📂 Struktur File

```
portfolio/
├── index.html          # Main HTML file
├── style.css           # Styling (Dark + Red theme)
├── script.js           # JavaScript functionality
├── README.md           # Documentation
├── catatan.md          # Notes
├── img/                # Images & photos
├── video/              # Video files
├── audio/              # Background music
└── icon/               # Icons & SVG files
```

## 🚀 Cara Menggunakan

1. **Clone atau download** repository ini
2. **Buka** `index.html` di browser
3. **Klik** icon musik (🎵) untuk memutar background music
4. **Navigasi** menggunakan sidebar menu
5. **Mobile**: Klik hamburger menu (☰) di kiri atas

## 📋 Sections

1. **Home** - Hero section dengan profile image
2. **About** - Perkenalan diri & hobi
3. **Education** - Riwayat pendidikan
4. **Skills** - Kemampuan teknis (HTML, CSS, JS, Canva, Figma, CapCut)
5. **Projects** - Portfolio project (web, video, poster)
6. **Tasks** - Dokumentasi kegiatan liburan
7. **Contact** - Informasi kontak & social media
